# Turno e Acoes

Cada rodada pode ser dividida em fases simples.

## Ordem sugerida

1. Movimento
2. Perigo do mar
3. Pesca obrigatoria
4. Mercado
5. Resolucao de efeitos pendentes

## Acoes base

- mover para outro mar;
- reparar casco;
- fortalecer o navio;
- jogar carta;
- vender peixes;
- guardar peixe para set futuro.

## Ideia complementar

Para o modo rapido, vale limitar o jogador a 1 decisao principal por turno. Isso reduz tempo de espera e evita turnos longos demais. No modo normal, essa camada pode ser expandida com mais interacoes e cartas de reacao.
