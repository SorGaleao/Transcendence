# Normal Mode

Future expansion mode, with deeper strategy.

## Goals

- preserve the longer-form identity of the game;
- include more tactical decision-making;
- better explore player conflict;
- increase the impact of cards and map positioning.

## Main rules

- more rounds;
- ship limits per sea (or an equivalent dispute system);
- direct combat enabled;
- wider card variety;
- more room for area control.

## Role of normal mode

It is not part of the current V1 baseline. It can be developed as a future expansion without changing the core system.

## Additional idea

If time gets tight, normal mode can be presented as a future layer or an alternative configuration of the same game engine.
