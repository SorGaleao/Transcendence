# Turn Flow and Actions

Each round can be divided into simple phases.

## Suggested order

1. Movement
2. Sea hazard
3. Mandatory fishing
4. Market
5. Pending effect resolution

## Core actions

- move to another sea;
- repair hull;
- reinforce ship;
- play a card;
- sell fish;
- hold fish for a future set.

## Additional idea

For quick mode, limit each player to one main decision per turn. This reduces downtime and avoids long turns. In normal mode, this layer can expand with more interactions and reaction cards.
